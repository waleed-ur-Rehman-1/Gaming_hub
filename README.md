# 🎮 Gaming Hub - Full Stack Gaming Platform

Gaming Hub is a full-stack web application that allows users to explore, download, and manage games through a modern and responsive interface. The platform provides game listings, detailed game information, secure user authentication, and an admin dashboard for managing content.

## 🚀 Features

### User Features
- Browse available games
- Search and filter games
- View game details and screenshots
- Download games directly
- User registration and login
- User profile management
- Responsive design for desktop and mobile

### Admin Features
- Admin dashboard
- Add new games
- Edit game information
- Delete games
- Manage users
- Upload game images and download files
